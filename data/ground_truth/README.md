# data/ground_truth/: known-good lineups (Phase 3)

30-50 community-known Ascent Sova lineups, one JSON file each, with **source attribution**. This set is the Phase 4 loss function, the permanent regression suite, and the project's falsifier (README §7).

Schema is finalized in Phase 3; it will include at minimum: approximate start position + stance, charge, bounce count, landing area + tolerance, source URL/timestamp, and, for several lineups per charge, **measured flight time with fps error bars** (required for identifiability; see math.md M-08).
